# MarketAdjuster

Professional market condition adjustments aligned with Fannie Mae guidance.

## Quick Install

### macOS
Open Terminal and paste:
```bash
curl -sL https://raw.githubusercontent.com/TKMAppraisals/MarketAdjuster/main/install_mac.sh | bash
```

### Windows
Open PowerShell and paste:
```powershell
irm https://raw.githubusercontent.com/TKMAppraisals/MarketAdjuster/main/install_windows.ps1 | iex
```
